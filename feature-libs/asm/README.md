# Spartacus ASM Library

Spartacus ASM can be added to the existing Spartacus application by running `ng add @spartacus/asm`. For more information about Spartacus schematics, visit the [official Spartacus schematics documentation page](https://help.sap.com/docs/SAP_COMMERCE_COMPOSABLE_STOREFRONT/31164ec95c7c4136b1d1a4a371cad3c7/e38d45609de04412920a7fc9c13d41e3.html).

For more information, see [Spartacus](https://github.com/SAP/spartacus).
